# QualeDoom - Versione Portable

## Come usare

1. Estrai questa cartella in qualsiasi posizione del tuo computer
2. Esegui `QualeDoom.exe` dalla cartella `QualeDoom-Portable`
3. Nessuna installazione richiesta!

## Avviso di sicurezza Windows

Quando esegui il programma per la prima volta, Windows potrebbe mostrare un avviso di sicurezza. Questo è normale per software non firmato:

### Soluzione 1 - Sblocco file (consigliata)
1. Tasto destro su `QualeDoom.exe` → **Proprietà**
2. Spunta **"Sblocca"** → **OK**
3. Riavvia il programma

### Soluzione 2 - SmartScreen
- Clicca su "Ulteriori informazioni" 
- Poi su "Esegui comunque"

### Soluzione 3 - Esecuzione come amministratore
- Tasto destro su `QualeDoom.exe` → "Esegui come amministratore"

## Requisiti

- Windows 10 o superiore
- GZDoom installato separatamente per avviare i WAD

## Caratteristiche

- ✅ Nessuna installazione necessaria
- ✅ Funzionamento da USB drive
- ✅ Non scrive nel registro di sistema
- ✅ Configurazioni salvate nella cartella del programma
